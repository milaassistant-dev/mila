Mila — это персональный ассистент для ПК, который объединяет голосовые команды, удаленное управление с телефона и быструю синхронизацию между устройствами в одном удобном приложении.

Первая публичная версия уже умеет закрывать рутину: от системных действий и горячих клавиш до обмена буфером и отправки скриншотов через аккаунтный сервер.

Что внутри
Голосовое и текстовое управление командами на ПК
Быстрые действия для окон, вкладок и системных сочетаний клавиш
Синхронизация буфера обмена между устройствами
Передача изображений и скриншотов через аккаунт
Интеграция с Telegram для удаленных сценариев
Локальные уведомления и удобный лог действий
Для кого
Для тех, кто хочет управлять ПК с телефона
Для тех, кто часто копирует текст и изображения между устройствами
Для тех, кому нужен компактный ассистент без лишней сложности
Статус релиза
Это первая публичная версия.
Фокус релиза: стабильная базовая функциональность и удобный ежедневный сценарий использования.
Дальше будет больше автоматизации, улучшения UX и расширение команд.
